# Savanyu

Don't mind this repository (private project).

## Dependancies
 
- `PHP >= 5.2`
- `Composer`
- `Free time`

## Building from git

```bash
$ git clone https://github.com/cisoun/Savanyu.git
$ cd Savanyu
$ mkdir vendor
$ chmod 777 -R vendor
$ chmod 777 -R storage
$ composer update
```

## Run

```bash
$ php -S localhost:8000 -t public
```
